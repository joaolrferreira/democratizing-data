JUPYTER NOTEBOOKS

Jupyter Notebooks can access metadata databases through SciServer, a platform supported by Johns Hopkins' IDIES. With a SciServer account and the right permissions, users can query databases using SQL syntax via a python library. Some databases are available for individual agencies and provide validated data without licensed snippet information. Access to these databases may vary depending on a user's access level.
